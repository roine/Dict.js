# Dict.js

## Introduction
Inspired by [Elm][elm] `Dict`, Dict.js simplifies the access and manipulation to array of objects. Dict.js does not mutate the collection instead it
returns a new collection.


## Installation
```bash
npm install @roine/dict --save
```
